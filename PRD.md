PRD.md 파일의 내용을 최우선으로 참고하여 프로젝트를 진행한다.

# 프로젝트 목표

대한민국 생활 법률 AI 챗봇의 AI 엔진을 개발한다.

현재 단계에서는 웹 서비스, 프론트엔드, 백엔드, API, 로그인, 회원관리 등은 절대 구현하지 않는다.

현재 목표는 EXAONE-4.0-1.2B와 RAG 시스템만 구축하는 것이다.

모델은 Hugging Face의 EXAONE-4.0-1.2B를 사용한다.

모델을 Fine-tuning 하지 않는다.

법률 데이터는 RAG(Retrieval-Augmented Generation) 방식으로 활용한다.

--------------------------------------------------

# 사용할 데이터

data/

├── TS_01. 민사법_001. 판결문/
└── TS_01. 민사법_002. 법령/

위 두 폴더의 모든 JSON 파일을 사용한다.

Validation 데이터는 사용하지 않는다.

--------------------------------------------------

# 구현 목표

## EXAONE

- Hugging Face에서 EXAONE-4.0-1.2B 모델 다운로드
- transformers 사용
- tokenizer와 model을 분리하여 관리
- CPU 환경에서도 실행 가능하도록 작성
- GPU(CUDA)가 존재하면 자동으로 GPU를 사용하도록 구현

--------------------------------------------------

## 데이터 전처리

- data 폴더를 자동 탐색
- 모든 JSON 자동 읽기
- JSON 구조 자동 분석
- 필요한 텍스트 자동 추출
- 공백 제거
- null 제거
- 중복 제거
- 메타데이터 유지

가능하면 아래 정보를 metadata에 저장

- 사건명
- 사건번호
- 법령명
- 조문번호
- 출처

--------------------------------------------------

## Chunk

긴 문서는 적절한 길이로 분할한다.

Chunk마다 metadata를 유지한다.

Chunk 크기는 config에서 수정 가능하도록 한다.

--------------------------------------------------

## Embedding

SentenceTransformer를 사용한다.

기본 임베딩 모델은

BAAI/bge-m3

를 사용한다.

모든 Chunk를 벡터화한다.

--------------------------------------------------

## Vector DB

FAISS를 사용한다.

생성된 Vector DB는

vector_db/

폴더에 저장한다.

이미 DB가 존재하면 다시 생성하지 않는다.

--------------------------------------------------

## Retriever

사용자 질문을 입력하면

관련 법령과 판결문 Top-5를 검색한다.

Top-K는 config에서 수정 가능하도록 한다.

--------------------------------------------------

## Prompt

검색된 법령과 판결문만 근거로 EXAONE가 답변하도록 Prompt를 작성한다.

반드시 아래 규칙을 포함한다.

- 검색 결과만 근거로 답변
- 추측 금지
- 관련 법령 표시
- 쉬운 한국어 사용
- 단계적으로 설명
- 일반적인 법률 정보이며 법률 자문이 아니라는 안내 포함

--------------------------------------------------

## 추론

CLI(chat.py)에서

질문 입력

↓

RAG 검색

↓

EXAONE 답변

까지 동작하도록 구현한다.

웹 UI는 절대 만들지 않는다.

# 개발 방식

절대로 한 번에 구현하지 않는다.

반드시 단계별로 진행한다.

각 단계가 끝나면

1. 구현 내용을 요약한다.
2. 생성하거나 수정한 파일을 설명한다.
3. 발생한 오류를 해결한다.
4. 추천 Git Commit 메시지를 작성한다.
5. 반드시 "다음 단계를 진행할까요?"라고 질문한다.

사용자의 승인 없이 절대 다음 단계를 진행하지 않는다.
